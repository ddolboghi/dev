# MPLS(Multiprotocol Label Switching)
## 목적과 역할
- IP 라우팅의 속도 개선
- 트래픽 엔지니어링 기능 제공
- 목적지 기반의 IP 라우팅을 대체하는 것이 아니라 보강하기 위한 기술이다.
- 선택적으로 데이터그램에 레이블을 지정하고 라우터가 목적지 IP 주소 대신 고정 길이 레이블을 기반으로 데이터그램을 전달할 수 있도록 한다.
- IP 주소 지정 및 라우팅을 사용하여 IP와 함께 작동한다.
