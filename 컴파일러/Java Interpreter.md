# 용어 정의
- 인수(argument): 함수 호출 시 실제로 함수에 전달하는 값이다. 따라서 함수 '호출'은 '인수' 리스트를 갖고 있다. actual parameter라고 부르기도 한다.
- 매개변수(parameter): 함수 본문 안에서 인수로 전달 받은 값을 담아둔 변수다. 따라서 함수 '선언'은 '매개변수'의 리스트를 갖고 있다. formal parameter라고 부르기도 한다.
- 클로저(closure): 로컬 함수(함수 안에 정의된 함수)가 자신이 사용하는 모든 주변 변수(surrounding variable, 해당 코드를 감싼 블록 바로 밖에 있는 변수)의 참조를 외부 함수가 리턴된 이후(함수가 리턴되는 순간에 로컬 변수는 사라짐)에도 계속 바라볼 수 있도록 어딘가에 보관하는 함수다. 함수가 주변 변수를 전혀 사용하지 않으면 클로저가 아니라 단순 중첩 함수일 뿐이다.
